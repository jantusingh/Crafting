☁️ Sincronizza tra dispositivi

Ufficio, casa, smartphone: stesso craft, stesse spunte (34/56), stesso magazzino e linee Smelt/Craft.
Come funziona: crea un Codice sync e inseriscilo uguale su tutti i dispositivi. Con il cloud attivo, ogni modifica si salva online e all'apertura si scarica l'ultima versione.
Cloud non configurato — usa Copia/Incolla oppure configura Firebase sotto.

Configurazione cloud (una volta sola)

Gratuito con Firebase. Crea un progetto → Firestore → incolla qui API Key e Project ID.
Regole Firestore (testo → Regole): match /crafting/{id}{ allow read, write: if true; } — il codice sync fa da password.

🌳 Totale per sbloccare tutto l'albero

Aggiungi un oggetto

Minerali (Fascia 1) = niente ricetta. Lingotti (Fascia 2) e Craft (Fascia 3) = aggiungi gli ingredienti.
Come funziona: Fascia 1 = minerali grezzi (estratti). Fascia 2 = lingotti (da minerali / lingotti). Fascia 3 = craft (da minerali / lingotti / altri craft). Un ingrediente può essere qualsiasi oggetto già creato.
